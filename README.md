Este aqui é um código mais experimental e artístico, misturando regressão linear com PyTorch, otimização por gradiente, aproximação de funções, e geração de imagens abstratas / arte computacional

Na parte de regressão / aprendizado, gerei retas aleatórias, visualizei várias regressões lineares, adicionei ruídos, e depois usei gradiente descendente para ajustar uma reta que tenta aproximar sin(x).

Na parte da arte generativa criei grades bidimensionais de coordenadasm usei funcções matemáticas como seno e transformei isso em imagens, e até usei rede neural não treinada para gerar uma imagem abstrata
